＜プログラムのメニュー登録＞
  「91 マスタ登録」、「101 システム管理マスタ」より
  「3002  統計帳票出力情報(月次)」に対して以下のプログラム登録の処理を行う。

  プログラム登録設定情報
    帳票番号      ：任意に設定
    帳票名        ：保留・再請求・月遅れ一覧表
    プログラム名  ：A21011A05
    帳票パラメタ１：出力区分  、FREE  (１・保留 ２・月遅れ ３・再請求 ９・全部)
    帳票パラメタ２：請求年月  、YM
    帳票パラメタ３：ＣＳＶ出力、FREE （1・出力、1以外は出力しない）
    
    【ＣＳＶファイル出力仕様】
     ・ファイル保存場所  /home/orca/
     ・ファイル名(規則)  A21011A05-#-YYYYMMDDhhmm.csv
                         (#="H":保留、="S":再請求、="T":月遅れ)
                         (YYYYMMDDhhmm=印刷日時)
     ・文字コードの形式  EUC-JP
