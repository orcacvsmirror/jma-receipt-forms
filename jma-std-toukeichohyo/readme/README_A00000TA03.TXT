＜インストール方法及び実行環境について＞
  １．ソースのコピーとコンパイル
      以下のインストールシェルをrootユーザで実行して下さい。
      bash install.sh

  ２．プログラムのメニュー登録
     「91 マスタ登録」、「101 システム管理マスタ」より
     「3002  統計帳票出力情報(月次)」に対して以下のプログラム登録の処理を行う。

     ＜プログラム登録設定情報＞
       帳票番号      ：任意に設定
       帳票名        ：未収金一覧表
       プログラム名  ：A00000TA03
       帳票パラメタ１：開始年月日  YMD （必須）  集計対象範囲の開始年月日を入力します。
       帳票パラメタ２：終了年月日  YMD （必須）  集計対象範囲の終了年月日を入力します。
       帳票パラメタ３：入外区分    FREE（必須）  1：入院  2：外来
       帳票パラメタ４：入金区分    N1  （任意）  空白：全てを対象  1：終了日（含む）以前を対象
