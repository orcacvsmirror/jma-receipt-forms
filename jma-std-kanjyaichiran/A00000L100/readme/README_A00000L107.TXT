＜はじめに＞
  このプログラムは指定日／指定期間において、預り金の患者の一覧を印字します。
  このプログラムは単体では動作しません。患者一覧表起動プログラムと連動して動作します。

  日医標準レセプトソフト(以下、日レセ)には標準で預り金は取り扱っていませんので
  預り金の取り扱いには預り金用の自費コードを用意して頂く必要があります。
  預り金用自費コードの用意、取り扱いに関しましては項目３「預り金の設定」にて説明致します。

＜インストール方法及び実行環境について＞
  １．ソースのコピーとコンパイル
      以下のインストールシェルをrootユーザで実行して下さい。
      bash install.sh

  ２．プログラムオプションのインストール
      初期インストールの場合、/prgoption/のディレクトリに移動し、
      次のインストールシェルをrootユーザで実行してください。

      bash  prgoption.sh <医療機関識別番号>
      例). bash  prgoption.sh 1

  ３．預り金の設定
      預り金については日レセに用意されていないため「点数マスタの自費コード」を
      １項目"預り金"として割り当て使用します。

      1)自費名称の設定
        「01 医事業務」-「91 マスタ登録」-「101 システム管理マスタ」-「1013 請求書自費名称」を選択。
        区分コードは任意の値を設定する。この値は2)でも使用します。
        有効期間は 00000000〜99999999
        自費名称を「預り金」として登録を行ってください。

      2)預り金用自費コードの作成
        「01 医事業務」-「91 マスタ登録」-「102 点数マスタ」を選択。
        診療行為コードは095から始まる任意の９桁コードを入力してください。

        例：095000001
        コードが入力できましたら、確定を押し画面を切替えてください。

        有効年月日「00000000〜99999999」
        カナ名称  「アズカリキン」
        漢字名称  「預り金」
        正式名称  「預り金」

        カラム位置、桁数はすべて0
        金額      「0」
        保険適用  「2 保険適用外」
        点数欄集計先識別(外来)   1)で設定した区分コードを入力してください。
        点数欄集計先識別(入院)   1)で設定した区分コードを入力してください。
        部位区分  「0 その他」

        入力チェック区分 「0」

        最後に登録を行ってください。

      3)患者一覧帳票への預り金用自費コードの登録
        2)で登録した預り金用自費コードを「91 マスタ登録」「101 システム管理マスタ」
       「1910 プログラムオプション情報」へ設定し登録してください。

  (注１) 日医標準レセプトソフト標準パッケージ実行環境に対するモジュールコピーは必要ありません。
  (注２) /usr/lib/jma-receipt/lddef/orcabt.bdファイルにプログラム情報を追加する必要はありません。
