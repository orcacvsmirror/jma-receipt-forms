1.はじめに

  このプログラムは指定年月において、各地区別に診療行為区分別の点数と回数を集計印字します。
  このプログラムは単体では動作しません。診療行為区分別集計表起動プログラムと連動して動作します。


2.インストール方法について

  このプログラムを次の手順でインストールします。

  ・ソースのコピーとコンパイル
  ・システムの再起動


   1)ソースのコピーとコンパイル
       プログラムのディレクトリへ移動し、
       次のインストールシェルをrootユーザで実行してください。

       sh install.sh

       実行するとソースのコピーを行った後にコンパイルを自動で行います。


   2)システムの再起動
       コンパイル作業終了後に必ずシステムの再起動(リブート)を実施して下さい。


   (注１) 日医標準レセプトソフト標準パッケージ実行環境に対するモジュールコピーは必要ありません。
   (注２) /usr/lib/jma-receipt/lddef/orcabt.bdファイルにプログラム情報を追加する必要はありません。


3.地区の設定について

  地区別に集計するためには./A00000K143/(該当のバージョン)/A00000K143.INIファイルに設定を書き込む必要があります。
  次の設定方法に基づいて、必要な地区情報を書き込んでください。

    1.地区名を登録する時は、サンプル地区と、「終了」の行との間に登録してください。
                             ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
    2.地区名の最大文字数は２０バイト(全角１０文字)です
    3.地区名が２０バイトに満たない場合は、残りを半角スペースで埋めてください。
    4.地区名と郵便番号との間には１バイトスペースを空けてください。
    5.１つの地区名に対して登録できる郵便番号の数は２０までです。
    6.登録出来る地区名はサンプルを除き、５０地区までです。
    7.郵便番号と郵便番号との間は、「,」で区切ってください。
    8.１行が181バイトになるようにしてください。181バイトに満たない場合は、必ず残りを半角スペースで埋めてください。
                                               ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
    9.最後の181バイト目は必ず改行コードになるようにしてください。
    10.郵便番号に*を使った場合は、その桁は全数字が有効数字となります。
         例 182****  =  1820000 〜 1829999 が対象
    11.ここに登録されていない郵便番号は「その他」の地区に分類されます。
